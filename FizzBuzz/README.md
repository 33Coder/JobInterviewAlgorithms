# FizzBuzz
**Problem Definition:**
This is a classic algorithm.
Task:
1. Print out numbers from 1-100.
2. Print out "FIZZ" for every 3rd number.
3. Print out "BUZZ" for every 5th number.
4. Print out "FIZZBUZZ" for every 15th number.

---

**Examples:**
* Output up to 5:
```
1
2
FIZZ
4
BUZZ
```
